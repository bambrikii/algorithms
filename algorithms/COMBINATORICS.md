1. [What is the difference between variation, combination, and permutations?](https://www.quora.com/What-is-the-difference-between-variation-combination-and-permutations)

### Permutations

```
P(n) = n!
```

### Variations

```
V(n, p) = P(n, p) = n! / (n! - p!)
```

### Combinations

```
C(n, p) = n! / p! /  (n - p)!
```
